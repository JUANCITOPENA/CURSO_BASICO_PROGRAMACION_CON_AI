===============================================================================================================================================

MODO DE SUBIR EL PROYECTO:

En la carpeta del proyecto en la página web, usted subirá una carpeta comprimida con el siguiente nombre:

NOMBRE_Y_APELLIDO_MATRICULA_PROYECTO_1_PORTAFOLIO_WEB.zip

Dentro de esta carpeta deben estar los archivos principales del proyecto:

index.html
style.css
script.js

PASOS PARA SUBIR EL PROYECTO A GITHUB:

Crear un Repositorio en GitHub:
Inicie sesión en su cuenta de GitHub.
Cree un nuevo repositorio con el nombre que se le haya indicado previamente (por ejemplo, nombre_apellido_portafolio_web).

Subir Archivos al Repositorio:

Comprima su carpeta del proyecto con el nombre especificado:

NOMBRE_Y_APELLIDO_MATRICULA_PROYECTO_1_PORTAFOLIO_WEB.zip .

En la página principal del repositorio, haga clic en "Add file" > "Upload files".
Arrastre y suelte los archivos (index.html, style.css, script.js) o cargue la carpeta comprimida.

Configurar GitHub Pages:

Vaya a la pestaña Settings en su repositorio.
Desplácese hasta la sección Pages .
Seleccione la rama main como fuente para GitHub Pages.

Guarde los cambios y copie la URL generada (por ejemplo, https://nombreusuario.github.io/nombre_apellido_portafolio_web).

Publicar y Desplegar el Proyecto:

Una vez configurado GitHub Pages, su proyecto estará disponible en la URL proporcionada.
Verifique que el sitio funcione correctamente accediendo a la URL desde un navegador.
